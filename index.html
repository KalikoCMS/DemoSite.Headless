Headless demo using React and Kaliko CMS

This demo project includes both a sample installation of Kaliko CMS as well as a small React application that consumes content through the content API. The main purpose is to show how to fetch content from a decoupled application.

The React application loads and renders CMS content from the start page as well as all articles published under the start page.

Start the project and access the React application by going to /react-app/build/index.html.

For more information

Find more information about this demo project over at GitHub.

Find more information about how to use Kaliko CMS as a headless CMS here.

If you have any questions, please post in the support forum.

Access admin for this demo project

You need to create a user before accessing the administration interface. This is done by opening SetupAdminAccount.aspx in the project root and follow the instructions given there.

Once the user has been created, go to /admin/

React source code

You'll find the source code for the React application in the \react-app\src\-folder of this project

To start the React development server by - standing in the \react-app\-folder - run:
npm start

Exploring the content API

Get all page types
/contentapi/v1.0/pagetype/all

Get all article pages
/contentapi/v1.0/pagetype/1/pages

Get the start page
/contentapi/v1.0/page/startpage

Get the start page by ID
/contentapi/v1.0/page/{id}

Get all children of the start page
/contentapi/v1.0/page/{id}/children
